# 地理探测器一条龙 V2.0 使用说明

## 1. 软件功能

`地理探测器一条龙V2.0.py` 是一个图形化处理工具，用于完成以下流程：

1. 按 shp 边界裁剪多个因子 tif。
2. 对裁剪后的因子进行自然断点五级分类。
3. 按用户设置的采样点间隔生成渔网点，提取 Y 值和 X 因子值。
4. 自动剔除存在空值或异常值的采样行。
5. 运行地理探测器，输出单因子探测表和交互探测热力图。

## 2. 启动方式

建议使用已有的 `Fenji` 环境运行，因为该环境已经包含 `rasterio`、`geopandas`、`jenkspy`、`matplotlib` 等依赖。

```bash
/opt/anaconda3/envs/Fenji/bin/python /Users/simon/PycharmProjects/Code/地理探测器一条龙/地理探测器一条龙V2.0.py
```

如果在 PyCharm 中运行，请把 Python 解释器设置为：

```text
/opt/anaconda3/envs/Fenji/bin/python
```

## 3. 输入数据

界面中需要设置以下路径：

| 参数 | 说明 |
| --- | --- |
| 因子 tif 文件夹 | 存放 X 因子的 tif 文件夹，例如 GDP、DEM、气温、降水等 |
| 边界 shp | 用于裁剪和生成采样点的研究区边界 |
| Y 栅格 tif | 因变量 Y，例如 NPP.tif |
| 裁剪输出文件夹 | 保存裁剪后的 tif |
| 五级分类输出文件夹 | 保存自然断点分类后的 tif |
| 结果输出文件夹 | 保存采样表、探测表、热力图等结果 |

默认路径已经按当前 Geo260 项目填写，可以直接检查后运行。

## 4. 参数说明

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| 分类级数 | 5 | 地理探测器常用五级分类，建议保持为 5 |
| Jenks 抽样上限 | 100000 | 大栅格计算自然断点时的抽样像元数，数值越大越慢 |
| 采样间隔(m) | 1000 | 渔网点间隔，默认 1000m x 1000m |
| 热力图色带 | 黄红渐变 YlOrRd | 可选择 YlOrRd、OrRd、inferno、magma、viridis、plasma、coolwarm 等 |
| 已有输出则跳过 | 勾选 | 若同名输出已存在，则跳过该步骤，避免重复计算 |

## 5. 推荐运行流程

首次处理建议按顺序点击：

1. `数据裁剪`
2. `五级分类`
3. `渔网提取 X/Y`
4. `地理探测器`

如果确认所有路径和参数无误，也可以直接点击：

```text
运行全部
```

## 6. 输出文件

结果输出文件夹中通常会生成：

| 文件 | 说明 |
| --- | --- |
| 五级分类自然断点.csv | 每个因子的自然断点阈值 |
| 渔网点XY提取值_1000m.csv | 当前采样间隔下的采样点坐标、Y 值和 X 因子值 |
| 单因子探测表.csv | 每个 X 因子的 q 值 |
| 交互探测q值矩阵.csv | 两两因子交互探测 q 值矩阵 |
| 交互探测类型矩阵.csv | 两两因子交互类型 |
| 交互探测热力图.png | 左下角半矩阵交互探测热力图 |
| 地理探测器样点摘要.txt | 采样间隔、有效样点数、X 因子列表等摘要 |

采样表文件名会带采样间隔。例如：

```text
渔网点XY提取值_1000m.csv
渔网点XY提取值_5000m.csv
```

这样更换采样间隔时，不会混淆不同尺度的采样结果。

## 7. 空值和异常值处理

软件在采样阶段会整行剔除以下情况：

1. Y 值为空、无穷大或极端异常值。
2. 任意 X 因子为空。
3. 任意 X 因子分类值不在 `1-5` 范围内。
4. 采样点落在栅格范围外或 NoData 区域。

地理探测器运行前会再次检查采样表，确保参与计算的数据为有效行。

## 8. 常见问题

### 采样点太多，运行较慢怎么办？

把 `采样间隔(m)` 调大。例如从 `1000` 改为 `2000`、`5000` 或 `10000`。

### 改了采样间隔后为什么没有重新生成采样表？

采样表文件名会包含间隔。如果当前间隔对应的采样表已经存在，并且勾选了“已有输出则跳过”，软件会直接复用该文件。需要强制重算时，可以取消勾选“已有输出则跳过”。

### 热力图颜色不好看怎么办？

在界面中选择其他热力图色带，然后重新运行 `地理探测器` 模块即可。

### 为什么要使用 Fenji 环境？

系统默认 Python 通常没有 GIS 依赖库。`Fenji` 环境中已经安装了本软件需要的栅格、矢量和绘图依赖。

## 9. 注意事项

1. 输入 tif 和 shp 可以坐标系不同，软件会在处理时自动重投影到合适坐标系。
2. 因子分类后的 X 值应为整数分层值。
3. 采样间隔越小，样点越多，计算越慢。
4. 大数据运行时请耐心等待，界面日志会显示当前进度。
5. 不建议在任务运行中关闭软件窗口。
